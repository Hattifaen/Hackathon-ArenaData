# Hackathon-ArenaData

```markdown
# 🔐 PII Scanner — Детектор персональных данных в изображениях


Автоматизированное решение для обнаружения персональных данных (ПДн) в изображениях форматов **TIF, PNG, JPG** в рамках хакатона «ИТ Академия-2026».

**Score на лидерборде: 10.857** 🏆

---

## 📁 Структура проекта

```
project/
│
├── Tif-Dif.py                     # 🚀 Основной исполняемый файл (финальная версия)
├── requirements.txt               # 📦 Python-зависимости
├── README.md                      # 📖 Этот файл
│
├── dataset/                       # 📂 ПАПКА С ДАТАСЕТОМ (создаётся пользователем)
│   └── share/                     # Распакуйте датасет сюда
│       ├── file1.tif
│       ├── file2.png
│       └── ...
│
└── output/                        # 📄 Результаты (создаётся автоматически)
    └── pdn_images_result.csv
```

## ⚙️ Требования

### Системные зависимости

| Компонент | Версия | Установка |
|-----------|--------|-----------|
| **Python** | 3.10 или выше | [python.org/downloads](https://www.python.org/downloads/) |
| **Tesseract OCR** | 5.x | [UB-Mannheim/tesseract](https://github.com/UB-Mannheim/tesseract/wiki) |

> ⚠️ **Критически важно:** При установке Tesseract **обязательно** выберите **русский язык** (`Russian`) в компонентах установки!

### Проверка установки Tesseract

```bash
tesseract --version
```

Должно появиться что-то вроде:
```
tesseract 5.3.3
```

Проверка наличия русского языка:
```bash
tesseract --list-langs | grep rus
```

## 🚀 Быстрый старт

### Шаг 1. Клонируйте репозиторий

```bash
git clone https://github.com/Hattifaen/Hackathon-ArenaData.git
cd <your-project-folder>
```

### Шаг 2. Установите Python-зависимости

```bash
pip install -r requirements.txt
```

### Шаг 3. Разместите датасет

1. Создайте папку `dataset/` в корне проекта (если её нет).
2. Скопируйте папку `share` с датасетом **внутрь** `dataset/`.

**Должно получиться так:**
```
project/
├── dataset/
│   └── share/           ← СЮДА РАСПАКОВАТЬ ДАТАСЕТ
│       ├── папка1/
│       ├── папка2/
│       ├── file1.tif
│       ├── file2.jpg
│       └── ...
```

### Шаг 4. Запустите сканирование

```bash
python Tif-Dif.py
```

## 🏗️ Архитектура решения

```
📁 СКАНЕР ФАЙЛОВ
    ↓
🖼️ ЭКСТРАКТОР (OCR)
    • PIL: grayscale, контраст, резкость, бинаризация
    • numpy: анализ качества → автовыбор метода
    • pytesseract: lang='rus+eng', PSM 3,6,11,13
    • ThreadPoolExecutor: 12 потоков
    ↓
🔍 ДЕТЕКТОР ПДн
    • 11+ regex-паттернов
    • Валидация: Лун, СНИЛС, ИНН, паспорт
    • Контекст ±75 символов
    • MD5-уникализация
    ↓
📊 КЛАССИФИКАТОР
    • Подсчёт по категориям
    • Группировка типов ПДн
    ↓
📄 ВЫВОД
    • CSV с колонкой 'name'
    • UTF-8-BOM для Excel
```
